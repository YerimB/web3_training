## Objective

1. Swap $ETH to $wETH (ERC20 version of $ETH)
2. Deposit some $wETH into Aave as collateral
3. Borrow $DAI with the deposited $wETH as collateral
4. Repay $DAI debt to Aave
5. Withdraw deposited $wETH
